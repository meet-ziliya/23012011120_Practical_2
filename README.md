# Practical-2: Android Activity Life Cycle Demo

This practical shows a basic Android app that displays “Hello World” and also demonstrates all the Activity Life Cycle methods using Log, Toast, and Snackbar.

---

## Objective
- Show **Hello World** in a TextView at the center of the screen.  
- Style the TextView (color, size, bold/italic).  
- Display messages for every Activity Life Cycle method using:
  - Log
  - Toast
  - Snackbar

---

## App Description
The app contains a single Activity with a centered “Hello World” TextView.

### TextView Details:
- Text: Hello World  
- Text Color: Holo Blue  
- Text Size: 27sp  
- Style: Bold + Italic  
- Layout Background: Yellow  

### Life Cycle methods used:
- onCreate  
- onStart  
- onResume  
- onPause  
- onStop  
- onRestart  
- onDestroy  

Each method shows a Log message, a Toast and a Snackbar so the flow of the Activity is easy to understand.

---

## Technologies Used
- Android Studio  
- Kotlin / Java  
- Android SDK  
- Material Components (Snackbar)

---

## How to Use
1. Download or clone the project.  
2. Open it in Android Studio.  
3. Run the app on a device or emulator.  
4. The Hello World message will appear at the center.  
5. Try rotating the phone, pressing Home, switching apps, etc.  
6. Check the Life Cycle messages in:
   - Logcat  
   - Toast  
   - Snackbar
